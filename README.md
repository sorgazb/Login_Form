> **Login Form** es un formulario de inicio de sesión responsivo diseñado con **HTML5** y **CSS3** puro, sin frameworks ni librerías externas. Ejemplo práctico de maquetación y estilado de formularios web modernos.

🔗 **[Prueba el formulario en vivo →](https://sorgazb.github.io/Login_Form/)**

---

## 📋 Descripción

El proyecto implementa un formulario de login con diseño limpio y moderno, haciendo uso único de tecnologías nativas del navegador:

- **HTML5** para la estructura semántica del formulario.
- **CSS3** para el diseño visual, efectos hover, transiciones y layout responsivo.
- **Sin dependencias externas**: cero frameworks, cero librerías.
- **GitHub Pages**: desplegado y accesible públicamente de forma inmediata.

## ⚙️ Instalación y Uso

Clona el repositorio:
```txt
git clone https://github.com/sorgazb/Login_Form.git
cd Login_Form
```

Abre directamente en el navegador:
```txt
open index.html
```

O usa la versión desplegada en **[GitHub Pages](https://sorgazb.github.io/Login_Form/)**.
